### 8.类
```Python
class student(object) :

    def __init__(self, id, name) :
        self.id = id
        self.__name = name

    def print_student(self) :
        print(self.id, self.__name)

a = student('01111295', 'Leo')
a.print_student()
try :
    print('id:', a.id)
    print('name:', a.__name)
except :
    print('error')
```
```Python
01111295 Leo 
id: 01111295 
error
```


### 9.继承

```Python
class fruit(object) :

    def __init__(self, name) :
        self.name = name

    def output_name(self) :
        print(self.name)


class apple(fruit) :

    def __init__(self, name) :
        self.name = name

a = apple("apple")
a.output_name()
type(a)
```
```Python
apple 
__main__.apple
```
